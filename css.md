# CSS
Responsável pela aparência visual do site  
Controla cores, tamanhos, espaçamentos, fontes e layout  
Funciona em conjunto com o HTML  

## Seletores
Seletor de elemento: aplica estilo a uma tag específica  
Seletor de classe: reutilização de estilos  
Seletor de id: estilização única  
Seletores combinados: relacionam elementos  
Seletores de atributo: aplicam estilo com base em atributos  

## Cores e fundos
color: cor do texto  
background-color: cor de fundo  
background-image: imagem de fundo  
background-repeat, background-size, background-position: controle do fundo  

## Texto e fontes
font-family: tipo da fonte  
font-size: tamanho do texto  
font-weight: espessura da fonte  
font-style: estilo da fonte  
text-align: alinhamento do texto  
text-decoration: decoração do texto  
line-height: altura da linha  
letter-spacing: espaçamento entre letras  

## Box Model (modelo de caixa)
width e height: tamanho do elemento  
padding: espaço interno  
border: borda do elemento  
margin: espaço externo  
box-sizing: controle do cálculo de tamanho  

## Layout e posicionamento
display: tipo de exibição do elemento  
position: posicionamento na página  
top, right, bottom, left: deslocamento  
z-index: ordem de sobreposição  
float e clear: alinhamentos antigos  

## Flexbox
Criação de layouts flexíveis  
flex-direction: direção dos itens  
justify-content: alinhamento horizontal  
align-items: alinhamento vertical  
gap: espaçamento entre itens  
flex-wrap: quebra de linha dos elementos  

## Grid Layout
Layout em linhas e colunas  
grid-template-columns e grid-template-rows  
gap: espaçamento entre células  
grid-column e grid-row: posicionamento dos itens  

## Espaçamentos e alinhamentos
margin e padding  
text-align  
vertical-align  

## Bordas e efeitos visuais
border-radius: cantos arredondados  
box-shadow: sombra externa  
opacity: transparência  

## Interações e estados
Pseudo-classes: :hover, :active, :focus, :visited  
Estilos aplicados a estados do elemento  

## Responsividade
@media: adaptação a diferentes telas  
Unidades relativas: %, em, rem, vw, vh  
Layouts flexíveis  

## Animações e transições
transition: suavização de mudanças  
animation: animações complexas  
@keyframes: definição de quadros de animação
